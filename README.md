# 休息提醒助手 🔔

一款帮助你养成规律休息习惯的桌面小工具。

---

## 功能特色

- **自定义间隔** — 设置工作多久后提醒一次（秒 / 分钟 / 小时）
- **自定义休息时长** — 设置每次休息的时间
- **自定义提醒文本** — 弹窗内容完全自定义
- **系统托盘** — 运行时缩到托盘，不占桌面空间
- **提醒弹窗** — 顶置显示，带倒计时进度条，可拖动
- **提前结束** — 休息弹窗支持"提前结束"和"跳过本次"
- **暂停 / 继续** — 随时暂停计时
- **今日统计** — 显示今天已完成的休息次数
- **设置持久化** — 参数保存到本地，下次启动自动加载
- **单实例保护** — 防止重复打开多个程序

---

## 快速开始

### 方式一：直接运行（需要 Python）

```bash
# 1. 安装依赖
pip install pyqt5

# 2. 运行
python main.py
```

### 方式二：打包为 EXE（Windows）

双击运行 `打包EXE.bat`，等待约 1-2 分钟，
生成文件位于 `dist\休息提醒助手.exe`，
双击即可使用，无需安装 Python。

---

## 使用说明

| 操作 | 说明 |
|------|------|
| **开始** | 启动计时，按钮变为"暂停" |
| **暂停** | 暂停当前倒计时，再次点击继续 |
| **设置** | 打开设置窗口（开始后置灰） |
| **退出** | 停止程序退出 |
| 点击关闭按钮 | 弹出选项：最小化到托盘 / 退出程序 |
| 最小化 | 程序最小化到系统托盘 |
| 双击托盘图标 | 恢复主窗口 |

## 关于安卓版本

如需安卓版，推荐使用 **Kivy + Buildozer** 方案：

1. 安装 Kivy：`pip install kivy`
2. 安装 Buildozer（需要 Linux / WSL 环境）
3. 运行 `buildozer android debug` 生成 APK

> 由于安卓版涉及通知权限、后台服务等系统级 API，
> 核心逻辑需要用 Kivy 的控件重写。
> 建议先用 PC 版验证功能，再考虑移植到安卓。
